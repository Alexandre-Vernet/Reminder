## Introduction

Reminder is a project that allows you to store notifications in a database and schedule their automatic delivery using cron-based reminders. Each notification can be configured with a specific date and time, name, description and the system handles automatic execution.

Features
- Store notifications in a database.
- Schedule notification reminders using cron jobs.
- Manage multiple notifications with different timings or frequencies.
